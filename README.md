# Media Horde API

## Installation

```
pip install -r requirements.txt  
```

## Run Server

```
python app.py
```

## Requests

All requests return a JSON object

### /AddGame

Create a new User

### /RemoveGame

Create a new Table

### /GetOtherGames

Add row to a Table

### /GetMyGames

Get all rows from a Table

### /GetAllGames

### /GetRandGames
